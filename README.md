## 🚀 Getting Started
If you want to explore or contribute, follow these steps:
1. **Clone the repository**
   ```bash
   git clone https://github.com/Yohannes-Alelign/python-practice.git
   ```
2. **Navigate into the directory**
   ```bash
   cd python-practice
   ```
3. **Run any Python script**
   ```bash
   python filename.py
   ```
